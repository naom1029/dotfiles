# tmux 設定

## Prefix

`Ctrl+g` (WezTermが`Ctrl+a`を使うため変更)

## キーバインド

### ウィンドウ（タブ）操作 - VSCode風

| キー | 動作 |
|------|------|
| `Ctrl+g c` | 新しいウィンドウ（タブ）追加 |
| `Ctrl+g n` | 次のウィンドウ（タブ）へ |
| `Ctrl+g p` | 前のウィンドウ（タブ）へ |
| `Alt+]` | 次のウィンドウ（prefix不要） |
| `Alt+[` | 前のウィンドウ（prefix不要） |
| `Alt+w` | ウィンドウを閉じる |

ステータスバーにタブ一覧が表示される（例: `[0:zsh] 1:zsh 2:zsh`）

### ペイン操作（画面分割用）

| キー | 動作 |
|------|------|
| `Alt+-` | 水平分割 |
| `Alt+\|` | 垂直分割 |
| `Alt+h/j/k/l` | ペイン移動 |
| `Alt+x` | ペインを閉じる |
| `Alt+\` | ズームトグル |

## 設定の反映

```bash
tmux source ~/.config/tmux/tmux.conf
```
