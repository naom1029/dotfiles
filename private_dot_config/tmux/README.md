# tmux 設定

## Prefix

`Ctrl+g` (WezTermが`Ctrl+a`を使うため変更)

## キーバインド

### ウィンドウ（タブ）操作 - VSCode風

| キー | 動作 |
|------|------|
| `Ctrl+g c` | 新しいウィンドウ（タブ）追加 |
| `Ctrl+g r` | 設定リロード |
| `Alt+]` | 次のウィンドウ（prefix不要） |
| `Alt+[` | 前のウィンドウ（prefix不要） |
| `Alt+w` | ウィンドウを閉じる（確認あり） |

ウィンドウ/ペイン番号は1始まり、閉じたら自動で詰まります。

### ペイン操作

| キー | 動作 |
|------|------|
| `Alt+-` | 水平分割（カレントディレクトリ継承） |
| `Alt+\|` | 垂直分割（カレントディレクトリ継承） |
| `Alt+h/j/k/l` | ペイン移動 |
| `Alt+x` | ペインを閉じる（確認あり） |
| `Alt+\` | ズームトグル |

### コピーモード（vi）

| キー | 動作 |
|------|------|
| `Ctrl+g [` | コピーモード開始 |
| `v` | 選択開始 |
| `Ctrl+v` | 矩形選択トグル |
| `y` | コピーしてモード終了 |

`tmux-yank` によりシステムクリップボードへ自動連携（OSC 52）。

## プラグイン（tpm 管理）

| プラグイン | 役割 |
|-----------|------|
| `tpm` | プラグインマネージャ |
| `tmux-sensible` | 妥当なデフォルト |
| `tmux-resurrect` | セッション保存/復元（pane内容含む） |
| `tmux-continuum` | 15分毎の自動保存・起動時自動復元 |
| `tmux-yank` | システムクリップボード連携 |

### プラグイン操作

| キー | 動作 |
|------|------|
| `Ctrl+g I` | プラグインのインストール |
| `Ctrl+g U` | プラグインの更新 |
| `Ctrl+g Alt+u` | 未使用プラグインの削除 |
| `Ctrl+g Ctrl+s` | セッションを手動保存 |
| `Ctrl+g Ctrl+r` | セッションを手動復元 |

## 設定の反映

```bash
tmux source ~/.config/tmux/tmux.conf
# または起動中なら  Ctrl+g r
```
