# WezTerm Configuration

このディレクトリには、Windows環境向けのWezTerm設定が含まれています。

## 📁 ファイル構成

```
.config/wezterm/
├── wezterm.lua      # メイン設定ファイル
├── keybinds.lua     # キーバインド定義
└── README.md        # このファイル
```

## ✨ 主な機能

### 基本設定
- **自動リロード**: 設定ファイルの変更を自動検出
- **日本語IME対応**: Windows環境で日本語入力をサポート
- **フォント**: Windows標準フォント（Cascadia Mono → Consolas → Courier New）
- **フォントサイズ**: 12pt

### UI設定
- **ウィンドウ透過度**: 0.85（85%）
- **タブバー**: 1つのタブのみの場合は自動非表示
- **カスタムタブスタイル**:
  - アクティブタブ: 黄色（#ae8b2d）
  - 非アクティブタブ: グレー（#5c6d74）

### シェル設定
- **デフォルトシェル**: PowerShell 7（WSL自動起動を回避）
- **WSL統合**: インストール済みWSLディストリを自動検出
- **Launch Menu**: PowerShell、cmd、WSLを選択可能

## ⌨️ キーバインド一覧

### リーダーキー

**リーダーキー**: `Ctrl + A` （tmux/screenと同じ）

リーダーキーを押した後、1秒以内に次のキーを押します。

---

### 🎯 基本操作

| キー | 説明 |
|------|------|
| `Ctrl + Shift + C` | コピー |
| `Ctrl + Shift + V` | ペースト |
| `Ctrl + Shift + P` | コマンドパレット |
| `Ctrl + Shift + R` | 設定リロード |
| `Alt + Enter` | フルスクリーン切り替え |

---

### 📑 タブ操作

| キー | 説明 |
|------|------|
| `Ctrl + T` | 新規タブ作成 |
| `Ctrl + W` | タブを閉じる |
| `Ctrl + Tab` | 次のタブへ |
| `Ctrl + Shift + Tab` | 前のタブへ |
| `Alt + 1~9` | タブ番号で直接切り替え（9は最後のタブ） |
| `Ctrl + A` → `{` | タブを左に移動 |
| `Ctrl + A` → `}` | タブを右に移動 |

### 🖼️ ペイン操作

#### ペイン分割
| キー | 説明 |
|------|------|
| `Ctrl + A` → `d` | 縦に分割（上下） |
| `Ctrl + A` → `r` | 横に分割（左右） |

#### ペイン移動（vim風）
| キー | 説明 |
|------|------|
| `Ctrl + A` → `h` | 左のペインへ |
| `Ctrl + A` → `j` | 下のペインへ |
| `Ctrl + A` → `k` | 上のペインへ |
| `Ctrl + A` → `l` | 右のペインへ |

#### その他のペイン操作
| キー | 説明 |
|------|------|
| `Ctrl + A` → `x` | 現在のペインを閉じる |
| `Ctrl + A` → `z` | ペインをズーム（全画面/解除） |
| `Ctrl + Shift + [` | ペイン選択モード |

#### ペインサイズ調整
| キー | 説明 |
|------|------|
| `Ctrl + A` → `s` | リサイズモードに入る |
| リサイズモード中: `h` | 左に縮小 |
| リサイズモード中: `j` | 下に拡大 |
| リサイズモード中: `k` | 上に拡大 |
| リサイズモード中: `l` | 右に拡大 |
| リサイズモード中: `Enter` | モード終了 |

#### ペイン移動モード
| キー | 説明 |
|------|------|
| `Ctrl + A` → `a` | ペイン移動モード（1秒間有効） |
| 移動モード中: `h/j/k/l` | ペイン間を移動 |

### 📋 コピーモード

**開始**: `Ctrl + A` → `[`

コピーモードでは、キーボードでテキスト選択ができます（vim風）。

#### 移動
| キー | 説明 |
|------|------|
| `h` / `j` / `k` / `l` | カーソル移動 |
| `w` | 次の単語へ |
| `b` | 前の単語へ |
| `e` | 単語の末尾へ |
| `0` | 行の先頭へ |
| `^` | 行の最初の文字へ |
| `$` | 行の末尾へ |

#### ジャンプ
| キー | 説明 |
|------|------|
| `f` + 文字 | その文字まで進む |
| `t` + 文字 | その文字の手前まで進む |
| `F` / `T` | 逆方向にジャンプ |
| `;` | 前回のジャンプを繰り返し |

#### スクロール
| キー | 説明 |
|------|------|
| `Ctrl + f` | 1ページ下へ |
| `Ctrl + b` | 1ページ上へ |
| `Ctrl + d` | 半ページ下へ |
| `Ctrl + u` | 半ページ上へ |
| `G` | 一番下へ |
| `g` | 一番上へ |
| `H` | 画面上部へ |
| `M` | 画面中央へ |
| `L` | 画面下部へ |

#### 選択モード
| キー | 説明 |
|------|------|
| `v` | 文字選択モード |
| `V` | 行選択モード |
| `Ctrl + v` | ブロック選択モード |
| `o` | 選択範囲の端を切り替え |

#### コピー・終了
| キー | 説明 |
|------|------|
| `y` | コピー |
| `Enter` | コピーして終了 |
| `Escape` / `q` / `Ctrl + c` | コピーモード終了 |

## 💡 使用例

### 例1: 画面を分割して複数の作業
```
1. Ctrl + A → d （縦分割）
2. Ctrl + A → r （横分割）
3. Ctrl + A → h/j/k/l （ペイン間を移動）
4. Ctrl + A → s （リサイズモード）
   - h/j/k/l でサイズ調整
   - Enter で終了
```

### 例2: WSLを起動
```
1. PowerShell が起動中（デフォルト）
2. Ctrl + A → d （新しいペインを分割）
3. Ctrl + Shift + P （コマンドパレット）
4. "Launch" で検索 → "WSL: Ubuntu" を選択
```

### 例3: ログをコピー
```
1. Ctrl + A → [ （コピーモード開始）
2. j/k でスクロール
3. v で選択開始
4. h/j/k/l で範囲選択
5. y でコピー（または Enter でコピーして終了）
6. Ctrl + Shift + V でペースト
```

### 例4: ワークスペースで環境を分ける
```
1. Ctrl + A → w （ワークスペース選択）
2. Ctrl + A → W （新規ワークスペース作成）
   - 例: "project-A", "monitoring" など
3. Ctrl + A → $ （ワークスペース名変更）
```

## 🔧 カスタマイズ

### フォントを変更したい場合

`wezterm.lua` の以下の部分を編集：

```lua
config.font = wezterm.font_with_fallback({
  'お好みのフォント',
  'Cascadia Mono',
  'Consolas',
})
```

### 透過度を変更したい場合

```lua
config.window_background_opacity = 0.85  -- 0.0（完全透明）～ 1.0（不透明）
```

### デフォルトシェルを変更したい場合

```lua
-- PowerShell 7
config.default_prog = { 'pwsh.exe', '-NoLogo' }

-- Windows PowerShell
config.default_prog = { 'powershell.exe', '-NoLogo' }

-- コマンドプロンプト
config.default_prog = { 'cmd.exe' }
```

### リーダーキーを変更したい場合

```lua
-- Ctrl+B にする場合（tmux風）
config.leader = { key = 'b', mods = 'CTRL', timeout_milliseconds = 1000 }

-- Ctrl+Space にする場合
config.leader = { key = 'Space', mods = 'CTRL', timeout_milliseconds = 1000 }
```

## 📝 メモ

- 設定ファイルは自動的にリロードされます
- エラーが発生した場合は、デバッグオーバーレイ（`Ctrl + Shift + L`）で確認できます
- WSLディストリは自動検出されるため、新しいディストリをインストールすると自動的にLaunch Menuに追加されます
